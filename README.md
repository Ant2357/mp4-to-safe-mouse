## Usage
```sh
git clone https://github.com/Ant2357/mp4-to-safe-mouse.git
cd mp4-to-safe-mouse

# 簡略化の為に書いてないけど、基本は venv 等使って
pip install -r requirements.txt
# 無修正の mp4 を安全な形に変換する
python main.py

```
